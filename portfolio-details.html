Afrijob Job Board Website

  • Category: Job Board Web Application
  • Tech Stack: Django (Backend), React (Frontend)
  • Project date: 01 March, 2024
  • Project URL: afrijob.com

Overview

Afrijob is a job board platform where job seekers can browse and search for posted jobs. Companies submit job details through a form, and an admin reviews and approves each job before it is displayed to job seekers. The backend is built with Django and the frontend with React.

Key Features
  • Admin approval workflow for job postings
  • Company job submission form
  • Job seeker job search and filtering
  • Modern, responsive UI

Abay Garment Ecommerce Website

  • Category: Ecommerce Web Application
  • Tech Stack: MERN (MongoDB, Express, React, Node.js)
  • Project date: 01 March, 2024
  • Project URL: abaygarment.com

Overview

Abay Garment is an ecommerce platform for selling garments online. Built with the MERN stack, it allows users to browse products, add items to their cart, and complete purchases securely.

Key Features
  • Product catalog and search
  • Shopping cart and checkout process
  • User authentication and order history
  • Responsive design for all devices
